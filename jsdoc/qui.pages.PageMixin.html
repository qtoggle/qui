Class: PageMixin

qui.pages.PageMixin(titleopt, nullable, pathIdopt, columnopt, keepPrevVisibleopt, modalopt, transparentopt, …args)

A mixin that offers page behavior.

Constructor

new PageMixin(titleopt, nullable, pathIdopt, columnopt, keepPrevVisibleopt, modalopt, transparentopt, …args)

Parameters:
Name Type Attributes Description
title String <optional>
<nullable>

page title
pathId String <optional>

identifies the page in the URL; leave this unset if the page should not be part of the URL or is the main page of a section
column Boolean <optional>

indicates that the page layout is a column and does not expand horizontally (defaults to false)
keepPrevVisible Boolean <optional>

indicates that the previous page should be kept visible while this page is the current one, to the extent possible (defaults to false)
modal Boolean <optional>

indicates that the page should be modal, not allowing any external interaction (defaults to false)
transparent Boolean <optional>

indicates that the page should be transparent (defaults to true)
args * <repeatable>
