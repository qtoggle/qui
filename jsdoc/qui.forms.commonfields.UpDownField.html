Class: UpDownField

qui.forms.commonfields.UpDownField(minopt, maxopt, stepopt, fastFactoropt, decimalsopt, continuousChangeopt, …args)

A field backed by an up-down (spinner) widget. The value data type is Number.

Constructor

new UpDownField(minopt, maxopt, stepopt, fastFactoropt, decimalsopt, continuousChangeopt, …args)

Parameters:
Name Type Attributes Description
min Number <optional>

the minimum value (defaults to 0)
max Number <optional>

the maximum value (defaults to 100)
step Number <optional>

the step value (defaults to 1)
fastFactor Number <optional>

determines the increment value when in fast mode (defaults to 10)
decimals Number <optional>

the number of decimals to display (defaults to 0)
continuousChange Boolean <optional>

set to false to prevent triggering change events with each value modificationwhen losing focus
