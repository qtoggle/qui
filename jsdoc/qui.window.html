(static) setSmallScreenThreshold(thresholdnullable)

Set the small screen threshold. Defaults to 700 logical pixels.

You can set the threshold to 0 to disable small screen mode; setting it to a large number (e.g. 1e6) ensures that small screen mode is always active.

This function may force-close current pages and re-navigate to the current path, if screen layout is changed.

Parameters:
Name Type Attributes Description
threshold Number <nullable>

small screen threshold, in logical pixels; passing null will reset to default
